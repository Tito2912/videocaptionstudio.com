Mode d’emploi — Sous-titres IA & TTS (Web/Desktop/Mobile)

Générer des sous-titres

  1. Ouvrez votre projet et placez le clip sur la timeline.
  2. Allez dans Texte → Sous-titres automatiques, choisissez la langue puis Créer.
  3. Éditez le minutage et le style (police, taille, couleur, effets). Option : piste bilingue.
  4. Export de la vidéo (sous-titres incrustés) ou export des fichiers de sous-titres si proposé.

Voix off IA (TTS)

  1. Collez votre script dans le module TTS.
  2. Choisissez une voix/un accent, réglez vitesse/hauteur, puis générez.
  3. Placez la narration sur la piste, alignez avec les coupes, exportez.

Astuce : pour les formats verticaux, combinez Auto Reframe + cuts dynamiques + styles de sous-titres brandés.

Workflows express pour TikTok/Reels/Shorts

  • Auto Reframe : dupliquez votre 16:9, convertissez en 9:16, gardez le sujet centré puis exportez.
  • Hook + Captions : mettez la phrase clé en ouverture ; style gras + emojis pour la rétention.
  • Plan B TTS : si l’audio est bruité, générez une TTS propre et mixez à −6 LUFS vs la musique.
